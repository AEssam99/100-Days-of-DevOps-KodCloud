# 🚀 100 Days of DevOps – Linux Tasks (Days 1–20)
# 📌 Overview
This repository contains the first 20 days of my 100 Days of DevOps Challenge, focused primarily on Linux system administration and foundational DevOps skills.


The goal of this phase is to build strong Linux fundamentals, including:

- User and permission management

- SSH configuration and security

- Process and service troubleshooting

- Networking and firewall configuration

- Web server and database setup

- Automation basics

This repository serves both as:

- 📖 A structured learning journal

- 🛠 A hands-on technical lab archive

- 🎯 A revision resource before interviews
# 🗂 Repository Structure
Each day has its own dedicated directory:
```sh
Day-01/
Day-02/
...
Day-20/
```
Inside each directory you will find:

- Task description

- Commands used

- Configuration steps

- Troubleshooting notes (if applicable)

- Each commit corresponds to a completed task for that specific day.
